## Data Cleaning methods used in the beginning -

-  Selected the headers with the help of `shift` [`Ctrl + shift` along with the arrow will also do] and filled them with a different color so that they can be distinguished from the rest of the data points.
  
-  Then I made sure that the column headers are in PROPER format. I did that first by injecting a helper row just below it. Then typed the `=PROPER(A1)` formula for the 1st cell. Post that to implement the same formula for the rest of the column headers I could have used the fill handle and drag it along the rows. However, it can be a cumbersome approach as there can be a large number of rows. Hence I selected the cells using `shift + arrow key` till the header cell, then I pressed `F2` + `Ctrl+Enter`. Voila! It has been implemented. Now I copied the cell values and pasted them on the header row and deleted the helper row.

- Then I removed double spaces from the whole dataset using the `Find and Replace` method, by simply replacing double spaces with single space.
